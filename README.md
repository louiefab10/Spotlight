# Spotlight 🎵

A personal music chart web app inspired by Korean music shows like Music Bank and Inkigayo. Connect your Spotify account to generate your own weekly chart based on your listening history.

## Features

- 🎤 Spotify OAuth login
- 📊 Personal chart based on your listening history
- ⏱ Three time ranges: This Month, 6 Months, All Time
- 🏆 Top 3 spotlight podium
- 🌙 Dark/Light theme toggle
- 🔗 Open any track directly in Spotify

## Tech Stack

- **Frontend:** React (Vite)
- **Backend:** Node.js, Express
- **Database:** MongoDB Atlas
- **API:** Spotify Web API
